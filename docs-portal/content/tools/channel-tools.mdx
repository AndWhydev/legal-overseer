# Channel Tools

Channel tools (`src/lib/agent/tools/channel-tools.ts`) provide unified access to all communication channels: Gmail, Outlook, WhatsApp, SMS, Slack, Apple Calendar, and Apple Reminders.

## Message Tools

### find_messages

Search across all connected channels for messages matching a query.

<ApiRefTable
  title="find_messages"
  props={[
    { name: "query", type: "string", required: true, description: "Sender name, subject keywords, content" },
    { name: "channel", type: "string", required: false, description: "Limit to: gmail, outlook, whatsapp, sms, slack" },
    { name: "from", type: "string", required: false, description: "Filter by sender name or email" },
    { name: "since", type: "string", required: false, description: "ISO date — how far back to look (default: 90 days)" },
    { name: "limit", type: "number", required: false, description: "Max results (default: 10)" },
    { name: "unread_only", type: "boolean", required: false, description: "Only return unprocessed messages" },
    { name: "folder", type: "string", required: false, description: "inbox, sent, or all (default: inbox)" },
  ]}
/>

<Tip>
Use `folder: "sent"` to find emails the user wrote. This reveals their voice, communication style, and relationship dynamics — essential context for drafting replies.
</Tip>
