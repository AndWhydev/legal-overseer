BitBit is an agentic AI operations platform that deploys autonomous agents to run business operations. Built on a Cognitive Memory OS architecture with 6 intelligence layers, BitBit combines a knowledge graph, proactive intelligence engine, and adaptive retrieval to deliver context-aware autonomous action.

## What BitBit does

BitBit operates as the intelligence layer of your business. It manages communications across email, SMS, WhatsApp, and iMessage. It tracks projects, generates invoices, and monitors client relationships. It learns from every interaction through a memory palace with episodic, semantic, and procedural memory types.

## Architecture at a glance

The system is built on 6 layers.

1. **Entity Graph Foundation** with pgvector for co-located graph and vector queries
2. **Graph-Aware Retrieval** replacing flat vector search with relationship-aware context
3. **Contextual Retrieval** enriching content at ingestion time for precision
4. **Sleep Consolidation** running nightly to summarize, resolve conflicts, and discover relationships
5. **Adaptive Query Routing** right-sizing retrieval per query complexity
6. **Predictive Loading and Procedural Memory** anticipating needs and automating learned workflows

## Core engine

At the heart is the TAOR loop (Think, Act, Observe, Repeat), an agentic decision engine that orchestrates tool execution, context assembly, and multi-step reasoning with confidence-based autonomy routing.
