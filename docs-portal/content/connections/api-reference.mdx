# Connections API Reference

Base URL: `https://app.bitbit.chat/api/connections`

## Authentication

| Endpoint | Auth method |
|----------|-------------|
| CRUD routes | Session cookie (Supabase auth) |
| Bridge ingest | Bearer token (`Authorization: Bearer {token}`) |
| Webhook | HMAC signature verification |
| Templates | No auth required |

---

## List connections

```
GET /api/connections
```

Returns all connections for the authenticated user's active organisation.

**Response:**
```json
{
  "connections": [
    {
      "id": "uuid",
      "org_id": "uuid",
      "provider": "outlook",
      "display_name": "Outlook",
      "transport": "bridge",
      "status": "connected",
      "message_count": 47,
      "last_sync_at": "2026-04-05T12:00:00Z"
    }
  ]
}
```

---

## Create connection

```
POST /api/connections
```

**Body:**
```json
{
  "provider": "my-crm",
  "display_name": "My CRM",
  "transport": "bridge",
  "template": "custom",
  "config": {}
}
```

Returns the created connection including generated `bridge_token` or `webhook_secret`.

---

## Get connection

```
GET /api/connections/{id}
```

---

## Update connection

```
PATCH /api/connections/{id}
```

---

## Disconnect

```
DELETE /api/connections/{id}
```

Soft deletes by setting `status: disabled`.

---

## Push messages (bridge)

```
POST /api/connections/{id}/ingest
```

**Headers:** `Authorization: Bearer {bridge_token}`

**Body:**
```json
{
  "messages": [{ "id": "...", "body": "..." }]
}
```

**Response:**
```json
{ "ok": true, "inserted": 5, "duplicates": 2 }
```

---

## Receive webhook

```
POST /api/connections/{id}/webhook
```

**Headers:** `x-webhook-signature: {hmac_sha256_hex}`

Accepts any JSON payload.

---

## Test connection

```
POST /api/connections/{id}/test
```

Inserts and removes a test message to verify the pipeline.

---

## Manual sync

```
POST /api/connections/{id}/sync
```

Triggers an immediate poll cycle (poll transport only).

---

## Sync logs

```
GET /api/connections/{id}/logs?limit=20&offset=0
```

**Response:**
```json
{
  "logs": [
    {
      "id": "uuid",
      "status": "success",
      "messages_found": 10,
      "messages_inserted": 8,
      "duplicates": 2,
      "duration_ms": 1200,
      "created_at": "2026-04-05T12:00:00Z"
    }
  ]
}
```

---

## List templates

```
GET /api/connections/templates
```

No authentication required.
