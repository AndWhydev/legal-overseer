Output: predictive-loader.ts integrated into context-assembler.ts.

**Signals to evaluate at conversation start:**
1. Deadline signal: entities with event_tuples where occurred_at within next 48h
2. Unresolved signal: entities with pending approval_queue items
3. Recency signal: entities with event_tuples in last 4h
4. Morning briefing: org settings morning_briefing data
</context>

<tasks>

<task type="auto">
  <name>Task 1: Build predictive-loader.ts</name>
  <files>src/lib/context-assembly/predictive-loader.ts</files>
  <action>
Export async function loadPredictiveContext(supabase, orgId):

1. Parallel fetch (Promise.allSettled):
   a. Deadlines: entity_nodes JOIN event_tuples WHERE occurred_at BETWEEN now() AND now()+48h
   b. Recent activity: entity_nodes JOIN event_tuples WHERE occurred_at > now()-4h
   c. Pending approvals: approval_queue WHERE status='pending' AND org_id=orgId, count
   d. Morning briefing: organisations.settings->morning_briefing

2. Collect unique entity_node IDs from all signals
3. For each entity (max 5): getNeighborhood for 1-hop context
4. Format as structured text block:
   "## Anticipated Context\n{deadline entities}\n{recent activity}\n{pending items}"
5. Return { text: string, entityNodeIds: string[], tokenEstimate: number }

Budget: shares the 1500-token proactive recall allocation. Predictive results are lower priority — they fill remaining space after entity-triggered recall.

Never throws, returns empty on failure.
  </action>
  <verify>TypeScript compiles.</verify>
  <done>predictive-loader.ts exports loadPredictiveContext(). Fetches signals in parallel, formats context.</done>
</task>

In the Phase 1 parallel fetch section of context-assembler.ts:

1. Import loadPredictiveContext
2. Add to the Promise.all alongside system prompt, recent messages, etc.
3. After proactive recall section: if proactive recall used < 1500 tokens, append predictive context to fill remaining budget
4. Inject as part of the system prompt (after proactive recall, before message history)

Pattern:
```typescript
const predictive = await loadPredictiveContext(supabase, orgId)
if (predictive.text && recallTokensUsed < 1500) {
  const remaining = 1500 - recallTokensUsed
  const trimmed = predictive.text.slice(0, remaining * 3.5)
  finalSystemPrompt += '\n\n' + trimmed
}
```

Also pass predictive.entityNodeIds to graphAwareRecall so it can include them in its recall.
