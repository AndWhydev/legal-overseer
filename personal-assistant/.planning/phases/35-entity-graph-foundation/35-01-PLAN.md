<task type="auto">
  <name>Task 1: Enable pgvector and create entity graph tables</name>
  <files>supabase/migrations/20260404000001_entity_graph.sql</files>
  <action>
Create migration that:

1. Enable pgvector extension: `CREATE EXTENSION IF NOT EXISTS vector WITH SCHEMA extensions;`

2. Create entity_nodes table:
   - id UUID PK default gen_random_uuid()
   - org_id UUID NOT NULL REFERENCES organisations(id) ON DELETE CASCADE
   - entity_type TEXT NOT NULL CHECK (entity_type IN ('person','project','company','invoice','channel'))
   - name TEXT NOT NULL
   - aliases TEXT[] DEFAULT '{}'
   - properties JSONB DEFAULT '{}'
   - embedding vector(768) — Google multimodal
   - text_embedding vector(1024) — Voyage-3.5
   - is_active BOOLEAN DEFAULT true
   - created_at TIMESTAMPTZ DEFAULT now()
   - updated_at TIMESTAMPTZ DEFAULT now()

3. Create entity_edges table:
   - id UUID PK default gen_random_uuid()
   - org_id UUID NOT NULL REFERENCES organisations(id) ON DELETE CASCADE
   - source_id UUID NOT NULL REFERENCES entity_nodes(id) ON DELETE CASCADE
   - target_id UUID NOT NULL REFERENCES entity_nodes(id) ON DELETE CASCADE
   - relation_type TEXT NOT NULL
   - properties JSONB DEFAULT '{}'
   - valid_from TIMESTAMPTZ NOT NULL DEFAULT now()
   - valid_until TIMESTAMPTZ — null means still true
   - ingested_at TIMESTAMPTZ DEFAULT now()
   - confidence FLOAT DEFAULT 0.8 CHECK (confidence >= 0 AND confidence <= 1)
   - source_memory_id UUID — nullable provenance link

4. Create event_tuples table:
   - id UUID PK default gen_random_uuid()
   - org_id UUID NOT NULL REFERENCES organisations(id) ON DELETE CASCADE
   - subject_id UUID NOT NULL REFERENCES entity_nodes(id) ON DELETE CASCADE
   - verb TEXT NOT NULL
   - object_text TEXT
   - object_id UUID REFERENCES entity_nodes(id) ON DELETE SET NULL
   - occurred_at TIMESTAMPTZ NOT NULL
   - occurred_until TIMESTAMPTZ
   - source_memory_id UUID
   - metadata JSONB DEFAULT '{}'
   - created_at TIMESTAMPTZ DEFAULT now()

5. Create indexes:
   - entity_nodes: GIN on aliases, btree on (org_id, entity_type) WHERE is_active, HNSW on embedding using vector_cosine_ops (m=16, ef_construction=64), HNSW on text_embedding using vector_cosine_ops
   - entity_edges: btree on (source_id) WHERE valid_until IS NULL, btree on (target_id) WHERE valid_until IS NULL, btree on (org_id, relation_type)
   - event_tuples: btree on (org_id, subject_id, occurred_at DESC), btree on (org_id, occurred_at DESC)

6. Enable RLS on all 3 tables. Create select/insert/update/delete policies matching the pattern in 002_rls_policies.sql (org_id = user's org from JWT).

7. Create updated_at trigger for entity_nodes (same pattern as contacts table).

Do NOT add foreign key to memory_palace_entries for source_memory_id — keep it as a soft reference UUID since memories may be archived.

<task type="auto">
  <name>Task 2: Create graph query TypeScript helpers</name>
  <files>src/lib/knowledge-graph/graph-queries.ts, src/lib/knowledge-graph/types.ts</files>
  <action>
Create TypeScript module for graph query operations:

1. types.ts — Export interfaces:
   - EntityNode (matches table schema)
   - EntityEdge (matches table schema) 
   - EventTuple (matches table schema)
   - GraphNeighborhood (node + edges + connected nodes)
   - GraphSearchOptions (org_id, entity_ids, relation_types, valid_at, limit)

2. graph-queries.ts — Export functions:
   - getEntityByAlias(supabase, orgId, alias): Find entity_node by alias match (case-insensitive, array contains)
   - getNeighborhood(supabase, orgId, entityId, opts?): 1-hop edges + connected nodes, filtered by valid_until IS NULL by default. Optional relation_type filter.
   - getMultiHopNeighborhood(supabase, orgId, entityId, depth=2, opts?): Recursive CTE for N-hop traversal. Cap depth at 3 to prevent explosion.
   - getEntityEvents(supabase, orgId, entityId, timeRange?): Event tuples for entity, ordered by occurred_at DESC.
   - vectorSearchEntities(supabase, orgId, queryEmbedding, topK=10): pgvector similarity search on entity_nodes.embedding using <=> operator, filtered by is_active=true.
   - findOrCreateEntity(supabase, orgId, name, type, aliases?): Upsert pattern — find by alias match first, create if not found. Returns entity_node.
   - createEdge(supabase, orgId, sourceId, targetId, relationType, props?): Insert edge, auto-invalidate any existing edge with same (source, target, relation_type) by setting valid_until=now().
   - createEventTuple(supabase, orgId, subjectId, verb, objectText, occurredAt, objectId?): Insert event tuple.

All functions use service-role supabase client (passed in). All filter by org_id. All handle errors gracefully (log + return null/empty, never throw).
