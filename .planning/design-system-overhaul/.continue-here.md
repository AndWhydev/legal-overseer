## Key Findings from Audit

### Glass Card Pattern (duplicated 40+ times across all temp files)
```css
background: rgba(10, 14, 23, 0.5);
backdrop-filter: blur(26px) saturate(1.15);
-webkit-backdrop-filter: blur(26px) saturate(1.15);
border: none;
box-shadow: inset 0 1px 0 rgba(255, 255, 255, 0.05);
border-radius: 20px;
```
Light theme variant:
```css
background: rgba(255, 255, 255, 0.65);
box-shadow: inset 0 1px 0 rgba(255, 255, 255, 0.8);
```

### Token Mismatches (globals.css :root vs design-system.css :root)
- `--text-dim`: design-system #64748B (fails AA at 3.9:1) → lighten to #7C8BA0
- `--text-muted`: globals.css #8A8279 vs not in design-system (different semantic)
- `--destructive`: globals.css #C47070, design-system --bb-red: #EF4444 (keep both, different roles)
- `--card`: globals.css #111113, design-system --bg-card: rgba(15,20,30,0.35) (keep both)
- `--color-bb-text-dim` in @theme block: #64748B needs update to match
