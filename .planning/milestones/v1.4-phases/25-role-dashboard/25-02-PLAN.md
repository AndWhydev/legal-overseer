# Plan 25-02: Autonomy Controls + Attention View

## Goal
Inline autonomy toggles per role and the "What needs my attention" unified view.

## Tasks
1. **AutonomyToggle component** — `src/components/roles/autonomy-toggle.tsx`: toggle Observer/Co-pilot/Autopilot per role. Calls PATCH /api/roles/[roleType]/autonomy.
2. **AttentionView component** — `src/components/roles/attention-view.tsx`: single screen showing all items requiring human input across all roles (pending approvals, escalations, high-priority insights).
3. **Attention API** — `src/app/api/roles/attention/route.ts`: GET items needing user attention, combined from approval_queue + role_activity escalations.
