# Plan 22-03: Comms Role State + Tests

## Goal
Define Comms-specific state schema, add overdue response escalation workflows, and write comprehensive tests.

## Tasks

### Task 1: Comms State Schema
Define CommsState interface in comms-role.ts:
- last_triage_at, last_followup_scan_at
- client_tone_profiles: Record<contactId, ToneProfile>
- engagement_baselines: Record<contactId, { avgMessagesPerWeek, lastCalculated }>
- active_followup_workflows: string[]

### Task 2: Overdue Escalation Workflow
Create `personal-assistant/src/lib/roles/comms/escalation-workflow.ts`:

### Task 3: Update Comms Index Exports
Update `personal-assistant/src/lib/roles/comms/index.ts` with all new exports

### Task 4: Tests

- Triage wrapper translation
- Follow-up detection with various SLA thresholds
- Engagement drop detection
- Tone adaptation
- Overdue escalation workflow
- Observer/Co-pilot/Autopilot routing
