# Plan 21-03: Cash Flow Monitoring + Payment Learning + Weekly Digest

## Goal
Finance role monitors cash flow (incoming/outgoing/pending), learns per-client payment patterns, and generates a weekly financial digest.

## Context
- Finance role operational from 21-01/21-02
- Existing: invoices table with status, amounts, dates, client contact
- Existing: bi_snapshots table for caching computed metrics
- Pattern: Supabase RPC functions for heavy SQL, results cached in bi_snapshots

## Tasks

### Task 1: Cash Flow Monitor
Create `personal-assistant/src/lib/roles/finance/cash-flow-monitor.ts`:

```typescript
export interface CashFlowSnapshot {
  period: 'current_month' | 'next_month' | 'next_quarter'
  incoming: number    // paid + expected from sent invoices
  outgoing: number    // known expenses (if tracked)
  pending: number     // sent but unpaid invoices
  overdue: number     // past due amount
  projectedBalance: number
  alerts: CashFlowAlert[]
}

export interface CashFlowAlert {
  type: 'shortfall_projected' | 'large_overdue' | 'payment_received' | 'unusual_delay'
  summary: string
  amount: number
  severity: 'high' | 'medium' | 'low'
}

export async function computeCashFlow(
  supabase: SupabaseClient,
  orgId: string,
): Promise<CashFlowSnapshot>
```

Queries invoices table grouped by status, computes projections, caches in bi_snapshots.
Generates alerts as role insights when shortfalls projected or large amounts overdue.

### Task 2: Payment Pattern Learning
Create `personal-assistant/src/lib/roles/finance/payment-learner.ts`:

```typescript
export interface PaymentPattern {
  contactId: string
  contactName: string
  avgPaymentDays: number
  medianPaymentDays: number
  onTimeRate: number     // % paid within terms
  totalInvoices: number
  lastPaymentAt: string | null
}

export async function learnPaymentPatterns(
  supabase: SupabaseClient,
  orgId: string,
): Promise<PaymentPattern[]>

export async function predictPaymentDate(
  supabase: SupabaseClient,
  orgId: string,
  contactId: string,
  invoiceDate: string,
  termsDays: number,
): Promise<{ predictedDate: string; confidence: number }>
```

Analyzes historical invoice payment data per client. Stores patterns in Finance role state.
Uses patterns to:
- Predict when an invoice will be paid
- Flag unusual delays ("Sezer usually pays in 7 days, it's been 16")
- Inform collection workflow timing

### Task 3: Weekly Financial Digest
Create `personal-assistant/src/lib/roles/finance/weekly-digest.ts`:

```typescript
export interface WeeklyFinanceDigest {
  period: { start: string; end: string }
  invoiced: { count: number; total: number }
  received: { count: number; total: number }
  overdue: { count: number; total: number }
  upcoming: { count: number; total: number } // due in next 7 days
  cashFlowSummary: string
  insights: string[]
  actionItems: string[]
}

export async function generateWeeklyDigest(
  supabase: SupabaseClient,
  orgId: string,
): Promise<WeeklyFinanceDigest>
```

Generated on weekly schedule (Monday morning). Delivered as:
- Dashboard card (always, all autonomy levels)
- Email summary (Co-pilot/Autopilot)
- WhatsApp summary (if configured)
