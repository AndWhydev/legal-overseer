**008_agent_configs.sql** — Agent registry and configuration.

Map from the `AgentConfig` TypeScript interface in `packages/core/src/types.ts`:

```sql
CREATE TABLE agent_configs (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  org_id uuid REFERENCES organizations ON DELETE CASCADE NOT NULL,
  agent_type text NOT NULL,  -- CHECK: same values as AgentType union
  name text NOT NULL,
  description text NOT NULL DEFAULT '',
  enabled boolean NOT NULL DEFAULT false,
  policy_rules jsonb DEFAULT '{}',
  channel_access text[] DEFAULT '{}',
  model_tier_override text CHECK (model_tier_override IN ('haiku', 'sonnet', 'opus')),
  confidence_thresholds jsonb DEFAULT '{}',  -- { act: 0.85, ask: 0.55 }
  notification_config jsonb DEFAULT '{}',  -- { channels: [...], targets: [...], escalation_delay_minutes: 15 }
  schedule jsonb,  -- { type: 'cron', cron_expression: '0 7 * * *' }
  created_at timestamptz DEFAULT now(),
  updated_at timestamptz DEFAULT now(),
  UNIQUE(org_id, agent_type)  -- one config per agent type per org
);
```

CHECK on agent_type: `IN ('lead-swarm', 'invoice-flow', 'channel-triage', 'client-comms', 'proposal-bot', 'ad-script-gen', 'client-onboarding', 'ai-search-optimizer', 'tender-hunter', 'sentry')`.

Indexes:

- `idx_agent_configs_enabled` on (org_id) WHERE enabled = true

Add updated_at trigger.

**009_agent_runs.sql** — Agent execution logging.

Map from `AgentRun` TypeScript interface:

```sql
CREATE TABLE agent_runs (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  org_id uuid REFERENCES organizations ON DELETE CASCADE NOT NULL,
  agent_config_id uuid REFERENCES agent_configs ON DELETE CASCADE NOT NULL,
  trigger_type text NOT NULL CHECK (trigger_type IN ('scheduled', 'webhook', 'manual', 'watch')),
  input_summary text NOT NULL DEFAULT '',
  output_summary text NOT NULL DEFAULT '',
  actions_taken jsonb DEFAULT '[]',  -- array of AgentAction objects
  tools_called text[] DEFAULT '{}',
  model_used text NOT NULL DEFAULT 'sonnet' CHECK (model_used IN ('haiku', 'sonnet', 'opus')),
  tokens_in integer NOT NULL DEFAULT 0,
  tokens_out integer NOT NULL DEFAULT 0,
  confidence_score float NOT NULL DEFAULT 0.0,
  routing_decision text NOT NULL DEFAULT 'escalate' CHECK (routing_decision IN ('act', 'ask', 'escalate')),
  duration_ms integer NOT NULL DEFAULT 0,
  error text,
  approved_by uuid REFERENCES auth.users ON DELETE SET NULL,
  approved_at timestamptz,
  created_at timestamptz DEFAULT now()
);
```

Indexes:
- `idx_agent_runs_config` on (agent_config_id, created_at DESC) -- runs for a specific agent
- `idx_agent_runs_org` on (org_id, created_at DESC) -- all runs for an org
- `idx_agent_runs_routing` on (org_id, routing_decision) WHERE routing_decision = 'ask' -- pending approvals

No updated_at trigger (runs are append-only, not updated -- except for approval fields, but those are one-time updates).

**010_leads.sql** — Lead pipeline for Lead Swarm agent.

Map from `Lead` TypeScript interface:

```sql
CREATE TABLE leads (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  org_id uuid REFERENCES organizations ON DELETE CASCADE NOT NULL,
  source_channel text NOT NULL,  -- 'gmail', 'whatsapp', 'website', 'referral', 'manual'
  source_detail text,  -- e.g. "Event Hero referral", "contact form"
  contact_id uuid REFERENCES contacts ON DELETE SET NULL,
  status text NOT NULL DEFAULT 'new' CHECK (status IN ('new', 'qualified', 'booked', 'converted', 'lost')),
  score text NOT NULL DEFAULT 'cold' CHECK (score IN ('hot', 'warm', 'cold')),
  budget_range text,  -- '$7-15k', '$500', etc.
  service_interest text[] DEFAULT '{}',  -- ['website', 'seo', 'ads']
  qualified_at timestamptz,
  converted_at timestamptz,
  notes text,
  metadata jsonb DEFAULT '{}',
  created_at timestamptz DEFAULT now(),
  updated_at timestamptz DEFAULT now()
);
```

Indexes:
- `idx_leads_org_status` on (org_id, status)
- `idx_leads_org_score` on (org_id, score)
- `idx_leads_contact` on (contact_id) WHERE contact_id IS NOT NULL

Add updated_at trigger.

**011_invoices.sql** — Invoice tracking for Invoice Flow agent.

Map from `Invoice` TypeScript interface:

```sql
CREATE TABLE invoices (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  org_id uuid REFERENCES organizations ON DELETE CASCADE NOT NULL,
  invoice_number text NOT NULL,
  client_contact_id uuid REFERENCES contacts ON DELETE SET NULL,
  status text NOT NULL DEFAULT 'draft' CHECK (status IN ('draft', 'sent', 'viewed', 'overdue', 'paid', 'cancelled')),
  items jsonb NOT NULL DEFAULT '[]',  -- array of InvoiceLineItem objects
  subtotal numeric(12,2) NOT NULL DEFAULT 0,
  tax numeric(12,2) NOT NULL DEFAULT 0,
  total numeric(12,2) NOT NULL DEFAULT 0,
  currency text NOT NULL DEFAULT 'AUD',
  issued_date date,
  due_date date,
  paid_date date,
  payment_method text,
  stripe_payment_link text,
  pdf_url text,
  sent_via text,  -- 'email', 'whatsapp'
  reminder_count integer NOT NULL DEFAULT 0,
  created_at timestamptz DEFAULT now(),
  updated_at timestamptz DEFAULT now(),
  UNIQUE(org_id, invoice_number)
);
```

Indexes:
- `idx_invoices_org_status` on (org_id, status)
- `idx_invoices_client` on (client_contact_id)
- `idx_invoices_due` on (org_id, due_date) WHERE status IN ('sent', 'viewed') -- find upcoming/overdue
- `idx_invoices_overdue` on (org_id) WHERE status = 'overdue' -- quick count of overdue

Add updated_at trigger.

**012_watches.sql** — Background monitoring watches for Sentry agent.

Map from `Watch` TypeScript interface:

```sql
CREATE TABLE watches (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  org_id uuid REFERENCES organizations ON DELETE CASCADE NOT NULL,
  agent_config_id uuid REFERENCES agent_configs ON DELETE SET NULL,
  watch_type text NOT NULL,  -- 'new_lead', 'invoice_overdue', 'negative_sentiment', 'no_response', 'custom'
  description text NOT NULL DEFAULT '',
  channel text,  -- 'gmail', 'whatsapp', etc. or null for cross-channel
  conditions jsonb NOT NULL DEFAULT '{}',  -- flexible: { threshold: 3, days: 7, keyword: "urgent" }
  interval_seconds integer NOT NULL DEFAULT 300,  -- 5 min default
  last_checked_at timestamptz,
  last_triggered_at timestamptz,
  status text NOT NULL DEFAULT 'active' CHECK (status IN ('active', 'paused', 'triggered', 'expired')),
  notification_targets jsonb DEFAULT '{}',  -- { channels: ['whatsapp'], targets: ['andy-user-id'] }
  created_at timestamptz DEFAULT now(),
  updated_at timestamptz DEFAULT now()
);
```

Indexes:
- `idx_watches_org` on (org_id, status) WHERE status = 'active' -- find active watches to poll
- `idx_watches_agent` on (agent_config_id) WHERE agent_config_id IS NOT NULL
- `idx_watches_due` on (org_id, last_checked_at) WHERE status = 'active' -- find watches due for check

Add updated_at trigger.
