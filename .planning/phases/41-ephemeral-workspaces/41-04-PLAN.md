# Plan 41-04: Isolation, Resource Limits + Integration Wiring

<objective>
Enforce network isolation between tenants, configure resource limits (CPU, memory, disk) at the provider level, wire workspace tools into the tool executor's budget system, and complete the end-to-end integration so workspace results flow from E2B through the TAOR loop back to the user. After this plan, ephemeral workspaces are production-ready with security and cost guardrails.
</objective>

<must_haves>
- Network isolation verified: workspace cannot access other tenants' resources
- Resource limits (CPU, memory) passed to E2B at sandbox creation
- Workspace tools integrated with tool executor budget system (TOOL_ROLE_MAP)
- Cost circuit breaker enforced before workspace creation
- Complete end-to-end flow: spawn -> exec -> output delivery -> destroy
- Orphan sweep integration point (callable from cron or health check)
</must_haves>

   ```typescript
   async spawn_ephemeral_workspace(input, orgId, supabase) {
     // Check workspace budget before creating
     const budget = await checkWorkspaceBudget(supabase, orgId)
     if (!budget.allowed) {
       return {
         success: false,
         error: `Daily workspace budget exhausted ($${budget.usedToday.toFixed(2)} of $${budget.limit.toFixed(2)} used). Try again tomorrow.`,
       }
     }

     const provider = new E2BProvider()
     const { sandboxId } = await provider.create({
       template: input.template || 'default',
       metadata: { orgId },
     })

     const session = await createWorkspaceSession(supabase, orgId, {
       sandboxId,
       purpose: input.purpose,
       template: input.template,
     })

     // Store provider reference for subsequent exec/destroy calls
     activeProviders.set(session.id, provider)

     return {
       success: true,
       data: {
         workspace_id: session.id,
         sandbox_id: sandboxId,
         status: 'running',
         message: `Workspace ready. Use workspace_exec to run code.`,
       },
     }
   }
   ```

   **`workspace_exec` handler** -- full implementation:
   ```typescript
   async workspace_exec(input, orgId, supabase) {
     const session = await getWorkspaceSession(supabase, input.workspace_id)
     if (!session || session.status !== 'running') {
       return { success: false, error: `Workspace ${input.workspace_id} not found or not running` }
     }

     const provider = activeProviders.get(input.workspace_id)
     if (!provider) {
       return { success: false, error: `Workspace provider not found. The workspace may have been reclaimed.` }
     }

     const execResult = await provider.exec(session.sandboxId, input.code, input.language)
     const delivered = await deliverWorkspaceOutput(supabase, input.workspace_id, execResult)

     return {
       success: !execResult.error,
       data: {
         output: delivered.inline,
         truncated: delivered.truncated,
         fullSizeBytes: delivered.fullSizeBytes,
         artifactIds: delivered.artifactIds,
         exitCode: execResult.exitCode,
       },
       error: execResult.error,
     }
   }
   ```

   **`workspace_destroy` handler** -- full implementation:
   ```typescript
   async workspace_destroy(input, orgId, supabase) {
     const provider = activeProviders.get(input.workspace_id)
     if (!provider) {
       return { success: false, error: `Workspace ${input.workspace_id} not found` }
     }

     await completeWorkspace(supabase, provider, input.workspace_id, 'completed')
     activeProviders.delete(input.workspace_id)

     return {
       success: true,
       data: { message: 'Workspace destroyed.' },
     }
   }
   ```
