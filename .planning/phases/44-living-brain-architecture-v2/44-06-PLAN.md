1. **computeDossierHash(markdown: string): string** — Returns SHA-256 hex of the markdown content using Node.js crypto: `createHash('sha256').update(markdown).digest('hex')`

2. **synthesizeDomainProfile(supabase, orgId: string, domain: DomainType): Promise<{ updated: boolean }>** —
   a. Load all entity dossiers for this org: `supabase.from('entity_dossiers').select('*').eq('org_id', orgId)` then filter by domain relevance (financial dossiers for financial domain, etc. — use signal_type mapping or include all for now)
   b. Compute current hashes: `{ [entityId]: computeDossierHash(dossier.dossier_markdown) }` for each dossier
   c. Load existing domain profile: `supabase.from('domain_profiles').select('*').eq('org_id', orgId).eq('domain', domain).single()`
   d. Compare constituent_hashes — if ALL hashes match, return `{ updated: false }` (skip re-synthesis)
   e. If any hash changed, synthesize via Sonnet/Opus:
      ```typescript
      const { text } = await generateText({
        model: models.balanced,
        system: `You synthesize a domain profile from multiple entity dossiers. Domain: ${domain}.
Output format:
## Overview
One paragraph summarizing the ${domain} state across all entities.
## Key Entities
Top entities with their current ${domain} status.
## Trends
Cross-entity patterns and trends.
## Risks
Any risks, warnings, or items requiring attention.
Keep under 3000 tokens.`,
        prompt: dossiers.map(d => `### ${d.entity_name}\n${d.dossier_markdown}`).join('\n\n'),
        maxTokens: 4000,
      })
      ```
   f. Upsert domain profile with new hashes, incremented version, updated token_count

3. **generateMorningBriefing(supabase, orgId: string): Promise<string>** —
   a. Load all 4 domain profiles for the org
   b. Synthesize into a morning briefing using Sonnet:
      - Top 5 priorities across all domains
      - Upcoming deadlines and risks
      - New discoveries or pattern changes
   c. Store briefing as a memory_palace_entry with category 'pattern' and source 'consolidation'

4. **startChiefLibrarian(supabase): Worker** — Creates worker on synthesis queue with concurrency 1. On each SynthesisJob:
   - Synthesize all 4 domain profiles
   - If any domain profile was updated, generate new morning briefing
