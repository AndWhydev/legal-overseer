     ```
     id uuid PK default gen_random_uuid()
     org_id uuid NOT NULL REFERENCES organizations ON DELETE CASCADE
     contact_id uuid REFERENCES contacts ON DELETE SET NULL (nullable -- not all sites are for a client)
     name text NOT NULL
     slug text NOT NULL (unique per org)
     description text
     template_id text (references built-in template slug, nullable)
     status text NOT NULL DEFAULT 'draft' CHECK (status IN ('draft','generating','preview','deployed','archived'))
     html_content text (the generated/edited full HTML)
     css_content text (optional separate CSS)
     metadata jsonb NOT NULL DEFAULT '{}'::jsonb (stores generation params, revision history refs, etc.)
     preview_url text (Supabase storage public URL or blob URL for staging)
     deploy_target jsonb (stores WP credentials/endpoint when linked: {type: 'wordpress', site_url, api_key_ref})
     deployed_at timestamptz
     deployed_url text
     created_at timestamptz DEFAULT now()
     updated_at timestamptz DEFAULT now()
     UNIQUE(org_id, slug)
     ```
   - Add indexes: `idx_website_projects_org` on (org_id), `idx_website_projects_status` on (org_id, status)

     id uuid PK default gen_random_uuid()

     created_by text (either 'user' or 'agent')
     created_at timestamptz DEFAULT now()
