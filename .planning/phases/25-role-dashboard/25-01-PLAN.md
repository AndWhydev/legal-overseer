# Plan 25-01: Role Activity Feed + Status Cards

## Goal
Unified role activity feed showing all role actions in priority order, and per-role status cards with key metrics.

## Tasks
1. **Role activity API** — `src/app/api/roles/activity/route.ts`: GET activity across all roles, sorted by priority/recency, filterable by role type.
2. **Role status API** — `src/app/api/roles/status/route.ts`: GET status for all roles (enabled, autonomy level, last tick, active workflows, key metrics).
3. **RoleActivityFeed component** — `src/components/roles/role-activity-feed.tsx`: unified feed of role actions, insights, escalations. Priority-sorted, not per-role silos.
4. **RoleStatusCards component** — `src/components/roles/role-status-cards.tsx`: per-role card showing state, active workflows, metrics, autonomy level.
