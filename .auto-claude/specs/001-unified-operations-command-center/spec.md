# Unified Operations Command Center

Create a centralized dashboard that merges inbox-driven events, lead/customer actions, and task state into a single operational view. Include filtering by workflow type, urgency, and agent ownership.

## Rationale
Primary users struggle with fragmented tools across inbox, CRM, and task management. A unified command center directly reduces context switching and improves execution consistency.

## User Stories
- As a small business operator, I want all operational work in one place so that I can run daily operations faster.
- As an operations manager, I want to triage by urgency so that high-priority items are handled first.

## Acceptance Criteria
- [ ] Given connected operational data sources, when a user opens the dashboard, then inbox, lead, and task items appear in one consolidated list/board.
- [ ] Users can filter and sort by status, assignee/agent, and priority within two interactions.
- [ ] Users can open item details and perform at least one primary action (assign, mark done, trigger workflow) without leaving the dashboard.
