The Solution

A swarm of specialized AI agents that act, not just recommend. Each agent owns a domain and operates autonomously within policies you define.

Confidence-Based Routing

ACT (>80%) — Auto-execute
ASK (50-80%) — Get approval
ESCALATE (<50%)

Agents aren't black boxes. Every action is auditable, policy-aware, and the confidence threshold is tuneable per agent.

Dual Purpose

For AWU (now)Andy gets his agency automated. Leads captured instantly. Invoices on time. Proposals write themselves. Client comms 24/7.
For BitBit (later)Every AWU deployment = real-world stress test. Edge cases become features. Battle-tested = market-ready SaaS product.

The 10 Agents Full Swarm

P0 Lead Swarm

Monitors all inbound channels, classifies leads, auto-acknowledges in <2 min, qualifies, books Calendly slots, scores (hot/warm/cold), escalates high-value to Andy.

  • Channels: Gmail, Outlook, WhatsApp, Facebook, Instagram, website forms
  • Never auto-rejects — uncertain leads escalate
  • Daily pipeline summary to Andy

P0 Invoice Flow

Auto-generates invoices from completed tasks, branded PDFs, sends via email, tracks payment status, automated reminder sequence.

  • Duplicate detection — never sends same invoice twice
  • Reminders: Day 1 (friendly), Day 7 (firm), Day 14 (final)
  • Stripe payment links + Xero/MYOB sync
  • Supports deposits, milestones, recurring

P0 Sentry

Background monitor using Gemini Flash (cheap). Watches for events and triggers other agents.

  • New lead from website → qualify + respond
  • Client replied to proposal → notify Andy
  • Invoice overdue → trigger reminder
  • Negative sentiment → escalate immediately

P1 Channel Triage

Pulls all channels into one prioritized feed. Classifies, deduplicates cross-channel, auto-creates tasks for actionable items.

  • Daily digest: "Here's what needs attention"
  • Thread tracking — who's waiting on who
  • Priority override: paying clients = minimum "high"

P1 Client Comms

Drafts replies in the correct voice (Andy's style, AWU brand, or client-specific). Weekly project updates. Sentiment analysis.

  • Per-client voice profiles and channel preferences
  • Meeting summary distribution after calls
  • Flags frustrated clients early

P1 Proposal Bot

Generates scope docs + pricing tiers from brief inputs or Cluely call transcripts. Professional PDFs. Follow-up sequences.

  • Target: brief to PDF in <30 min (currently 2-4 hrs)
  • Auto-includes relevant case studies
  • Contract generation from accepted proposals
  • Uses Opus (complex reasoning)

P2 Ad Script Generator

Generates 30+ video scripts/month for social ads. Hook variations, platform-adapted (Reels, TikTok, Shorts, Feed).

  • Ingests offer packages + competitor research
  • A/B hook variants per core script
  • Storyboard generation with shot descriptions
  • Performance feedback loop — learns what converts

P2 Client Onboarding

Triggers on deal acceptance. Auto-creates Asana project, sends welcome pack, requests credentials, books kickoff call.

  • Templates by project type (web, SEO, ads, AI)
  • Sets up GSC + Analytics access
  • Tracks onboarding completion checklist

P2 AI Search Optimizer

Audits visibility on ChatGPT, Gemini, Perplexity, Claude. Generates AI-optimized content. $2k/month service opportunity.

  • Low competition — most agencies haven't caught on
  • Australian market particularly underserved
  • First-mover advantage
  • Schema + entity markup optimization

P3 Tender Hunter

Monitors AusTender, QTenders, NSW eTendering. Filters by AWU capabilities. Auto-extracts requirements. Drafts responses.

  • Scores tender fit (effort vs. contract value)
  • Compiles required docs (ABN, insurance, case studies)
  • Builds template library over time
